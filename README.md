# 檔案命名格式說明
## 美術素材
### 動畫
Anim_角色名稱_素材名稱_編號\
\
**範例**\
Anim_Duck_Attack1st_001\
\
\*若不是逐格則不用編號\
Anim_Duck
### 背景
BG_素材名稱_類別(far、medium、forward、object)\
\
**範例**\
BG_meadow_forward
### 立繪
Portrait_角色_差分(表情)_編號\
\
**範例**\
Portrait_Aldonza_happy_001
\
\*若只有一張就不用編號\
Portrait_DuckNude_normal
### 劇情演出(CG、漫畫的部分)
CG/Comic_素材名稱_編號\
\
**範例**\
Comic_openning_001
\
\*若只有一張就不用編號\
CG_HappyEnding
### UI
UI_功能_素材名稱\
**範例**\
UI_HpBar_background
## 音樂素材
Music_應用場景_素材名稱_檔案種類_版本號
\
**範例**\
Music_battlefield_Test_Loop_v1
## 音效素材
SFX_應用場景_素材名稱\
\
**範例**\
SFX_Final_Victory
